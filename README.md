# Sales Pitch Embeds

Animated interactive HTML embeds for Huddle Duck's Notion sales presentations. Each file is a standalone HTML page designed to be embedded in Notion via `/embed` blocks.

## Files

| File | Purpose |
|---|---|
| `hero-header.html` | Hero banner with animated headline |
| `step-01-avatar.html` | Step 1: Avatar/targeting explanation |
| `step-02-strategy.html` | Step 2: Strategy overview |
| `step-03-content.html` | Step 3: Content creation process |
| `step-04-targeting.html` | Step 4: Ad targeting approach |
| `step-05-recycling.html` | Step 5: Content recycling system |

## Features

- Looping CSS animations with premium feel
- AI/Human ratio slider (interactive power bar with shine effect)
- Fully self-contained - no external dependencies
- Designed for dark backgrounds in Notion embeds

## Usage

1. Host the HTML files (GitHub Pages, Vercel, or any static host)
2. In Notion, type `/embed` and paste the hosted URL
3. Resize the embed block to fit your page layout

Each file works independently - no shared CSS or JS required.
